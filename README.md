# Project67 — CloudStream3 Extensions

Kumpulan extension CloudStream3 buatan sendiri.

## Install

Klik tombol di bawah untuk langsung menambahkan repo ke CloudStream3:

[![Install di CloudStream3](https://img.shields.io/badge/CloudStream3-Install%20Repo-orange?style=for-the-badge)](cloudstreamrepo://raw.githubusercontent.com/cemmekx096-cmd/Project67/refs/heads/main/repo.json)

Atau tambahkan URL repo secara manual di CloudStream3:

```
https://raw.githubusercontent.com/cemmekx096-cmd/Project67/refs/heads/main/repo.json
```

**Cara manual:**
1. Buka CloudStream3
2. Settings → Extensions → + Add Repository
3. Paste URL di atas → OK
